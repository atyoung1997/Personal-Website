Project Details: IMDB Movie Ratings Analysis

Project Summary

The dataset for this project contains a listing of the top 1000 movies on IMDB based on user ratings. I wanted to explore what makes a movie more likely to make it to the top 1000, and how the IMDB user ratings differ from Metacritic's metascore, which is a rating based on reviews from a curated list of critics.

Key Takeaways

  • "Drama" was by far the most common tag for movies in the IMDB top 1000. My theory is that drama is the most universal tag, meaning that it can easily apply to an action movie, a comedy movie, or really any type of movie.

  • The sweet spot for movie-length is 1.5-2.5 hours.

  • The Metacritic metascore system rated older movies much higher and newer movies much lower than the IMDB user ratings.

  • IMDB Users tended to enjoy sports, action, crime, sci-fi and fantasy movies much more than the metascore critics.
