Project Details: Outdoor Image Classification

Project Summary

The dataset for this project contains basic info (title, release year, genre, publisher, and console) for ~16,500 video games released between 1985 and 2017. My main goal for this project was to see if these basic characteristics could create a model with reasonable predictive power. Initially I had hoped to use regression techniques to predict a continuous global sales variable, but it quickly became clear that the independent variables were not powerful enough to create a useful model. With this in mind, I turned the project into a classification problem, and attempted to classify whether a videogame would sell >500K copies. I also decided to experiment with creating all custom exhibits for the presentation with simple shapes and charts.

Key Takeaways

  • Predicting the success of a video game is a complex process, and is clearly a result of factors outside of the scope of this dataset. If you've played even a single video game in your life, you'll recognize that it's success is not a function of broad characteristics like it's genre or publisher, but rather of minute details like the smoothness of it's gampeplay, the quality of it's story, and the polish of it's user interface. These factors simply can't be quantified in a dataset.

  • In general, categorical independent variables are not very useful if each category has a high level of variation with respect to the dependent variable.
