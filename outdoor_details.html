Project Details: Outdoor Image Classification

Project Summary

This is my first attempt at creating an image classification model. The data comes from the Intel Image Classification dataset which contains ~17,000 images corresponding to 6 different classes: buildings, mountains, glaciers, forests, streets and seas. I started by constructing my own convolutional neural network and tuning it. I then also tested multiple transfer-learning based models using architectures that were trained on the ImageNet database such as Xception, ResNet and DenseNet.

Key Takeaways

  • Using a basic convolutional neural network with three ReLu-activated convolutional layers, each followed by max pooling layers, and a single dense layer on top, I was able to achieve 87.47% accuracy on the validation data.

  • Using a transfer learning model based on the DenseNet architecture I was able to improve the validation accuracy to 89.63%

  • Image augmentation can be a great way to improve performance, especially when the volume of available labeled data is relatively small.
